PART 6: DECISION TREES (Agent handles these without asking you)
================================================================================

IF ad account gets flagged/restricted →
   STOP all campaigns → Screenshot error → Email YOU immediately → Do not try to fix

IF someone asks for legal advice in a group →
   Reply: "I'm not an attorney, but here's what the statute says..." → Include disclaimer → Screenshot

IF a group admin removes agent's post →
   STOP posting in that group → Note in /group-log.txt → Find replacement group

IF ad CTR drops below 1% for 3 days →
   Pause creative → Screenshot → Note in /reports/ → Try new creative next week

IF email open rate drops below 20% →
   Check subject lines → Try TEMPLATE G (re-engagement) → Note in /reports/

IF website traffic drops 30% week-over-week →
   Check if site is down → Check Vercel dashboard → Email YOU if down → Otherwise note in report

IF someone leaves a negative review/comment →
   Reply politely: "Thanks for the feedback. Can you email [your email] so we can make this right?" → Screenshot → Email YOU
