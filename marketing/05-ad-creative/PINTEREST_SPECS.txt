CANVA TEMPLATE STRUCTURE (Duplicate this for each state)
--------------------------------------------------------------------------------
Layer 1: Background color based on grade
  A = light green (#D1FAE5)
  B = light yellow (#FEF3C7)
  C = light orange (#FFEDD5)
  D/F = light red (#FEE2E2)

Layer 2: State name (large, bold, top 30%)
  Font: Montserrat Bold, 72pt
  Color: #1E293B

Layer 3: Grade letter (HUGE, center)
  Font: Montserrat Black, 200pt
  Color: Grade color (green/yellow/orange/red)
  Shadow: subtle drop shadow for depth

Layer 4: Score (below grade)
  Font: Montserrat SemiBold, 48pt
  Color: #1E293B
  Text: "[X]/100"

Layer 5: 4 mini-bars (below score)
  Each bar: 200px wide, 12px tall, rounded
  Colors: fill = grade color, background = #E5E7EB
  Labels: "Reporting [X]", "Testing [X]", "Curriculum [X]", "Teacher [X]"
  Font: Inter Regular, 14pt

Layer 6: CTA button (bottom 15%)
  Background: #3B82F6, rounded corners
  Text: "See [State] Requirements →"
  Font: Inter SemiBold, 18pt, white

Layer 7: Logo/watermark (bottom right corner, small)
  "Homeschool Compass" + compass icon
  Font: Inter Regular, 12pt, #6B7280
